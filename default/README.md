# Math Operations

This repository contains basic math operations implemented in Python. The available functions are:

- `add(a, b)`: Adds two numbers and returns the result.
- `subtract(a, b)`: Subtracts the second number from the first and returns the result.

## Workflow

This repository is integrated with a CI pipeline to ensure code quality and correctness. The pipeline automatically runs tests on every push and pull request to the `main` branch.

## Usage

1. Clone the repository.
2. Navigate to the `src` folder to find the `math_operations.py` file.
3. Run the functions as needed.

## Testing

Tests for the functions are located in the `tests` folder. To run the tests, execute:

```bash
pytest tests/
```
