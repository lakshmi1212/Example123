# Math Operations

This repository contains basic mathematical operations and their corresponding test cases.

## Usage

- **Addition**: Use the `add(a, b)` function to add two numbers.
- **Subtraction**: Use the `subtract(a, b)` function to subtract two numbers.

## Workflow

1. Clone the repository.
2. Install the dependencies from `requirements.txt`.
3. Run the test cases using `pytest`.

## CI/CD Workflow

This repository includes metadata for generating CI/CD workflows for automated testing and deployment.
